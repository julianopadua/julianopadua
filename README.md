## Projetos de Pesquisa e Desenvolvimento

**Forest Portal**  
Infraestrutura de dados abertos integrando fontes da CVM, INPE e EIA. Construída com Next.js e Supabase, o projeto centraliza indicadores ambientais e financeiros para análise pública.

**Análise de Focos de Incêndio no Cerrado - TCC**  
Pesquisa sobre o desempenho comparativo de modelos de Machine Learning (XGBoost, Random Forest) para previsão de queimadas no bioma Cerrado. O fluxo de trabalho é centrado em Python, TypeScript e modelagem preditiva.
